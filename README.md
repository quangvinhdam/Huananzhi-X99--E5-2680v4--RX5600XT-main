# Sonoma On Xeon X99 - High-End Desktop (HEDT) Broadwell-E

-	CPU: 2x 14-Core Intel Xeon E5-2680 v4, 2400 MHz (24 x 100)
- Motherboard: Huananzhi X99-F8D

- Audio Adapter: Realtek ALC887 (Work with AppleHDA)
